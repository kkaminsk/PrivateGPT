Step 1: Install Foundry Local

Foundry Local is Microsoft's runtime for running AI models locally on your device. Run this command in PowerShell or Windows Terminal:

PowerShell / Terminal
winget install Microsoft.FoundryLocal

Don't have winget? Download manually

Step 2: Download a Model

Now download an AI model. This command will download and cache the phi-3-mini-4k model (about 2.4 GB). Run this in PowerShell or Terminal:

PowerShell / Terminal
foundry model run phi-3-mini-4k

Note: The download may take several minutes. Wait for it to complete in the terminal before checking.

Setup Complete!

Foundry Local is installed and you have models ready to use.

Foundry Local Required

PrivateGPT requires Foundry Local to run AI models on your device. It appears Foundry Local is not installed.

Download and install from:

https://github.com/microsoft/Foundry-Local/releases

Install using PowerShell:

Add-AppxPackage foundry-local.msix
